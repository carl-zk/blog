Trie树(字典树)

发表于 2018-03-31 更新于 2024-05-18 分类于 数据结构

字典树就是根据字母查单词，假设全都是小写字母，
根节点的 key = ‘ ‘，在第一层;
第二层最多26个节点，即 key = a 到 z;
第三层最多26^26个节点，以此类推，最底层的高度为最长单词的长度。
