жҲ‘ж №жҚ®еҜ№ Pipeline зҡ„зҗҶи§ЈпјҢз„¶еҗҺе°қиҜ•е°ҶдёҖдёӘ Spring Boot Web йЎ№зӣ®зј–иҜ‘пјҢжү“еҢ…пјҢз”ҹжҲҗй•ңеғҸе’ҢйғЁзҪІгҖӮж•ҙдёӘдёӢжқҘжүҚеҸ‘зҺ°еҺҹжқҘжҲ‘дёҖзӣҙд»ҘдёәиҮӘе·ұзҗҶи§Јд»Җд№ҲжҳҜж»ҡеҠЁеҸ‘еёғпјҢдҪҶжҳҜе®һйҷ…дёҠжҲ‘дёҚжҮӮеҰӮдҪ•з”Ё Jenkins еҒҡеҲ°гҖӮ

Pipeline

https://jenkins.io/doc/book/pipeline/

Pipeline жҳҜдёҖжқЎжөҒж°ҙзәҝпјҢж¶өзӣ–дәҶж•ҙдёӘйЎ№зӣ®д»Һзј–иҜ‘гҖҒжү“еҢ…еҲ°жөӢиҜ•гҖҒйғЁзҪІдёӯй—ҙжүҖжңүзҡ„жӯҘйӘӨгҖӮдёҖиҲ¬еңЁJenkinsfileж–Үд»¶дёӯе®ҡд№үж•ҙдёӘ pipelineгҖӮ

е…ҲзҗҶи§Ј Pipelineзҡ„еҮ дёӘжҰӮеҝөгҖӮ

  • Node
    Jenkins service зҡ„дёҖйғЁеҲҶпјҢеҸҜд»Ҙи·‘ Pipeline зҡ„жңәеҷЁгҖӮJenkinsfileдёӯ agent { node { label вҖҳlabelNameвҖҷ } } еҸҜд»ҘжҢҮе®ҡ nodeгҖӮеҲҡиө·еҠЁзҡ„ Jenkins еҸӘжңү master дёҖдёӘиҠӮзӮ№гҖӮ
  • Stage
    иҮӘе®ҡд№үдёҖз»„ taskпјҢеҸҜеңЁ Jenkins UI зңӢеҲ° status/progressгҖӮ
  • Step
    дёҖдёӘ taskпјҢеӯҳеңЁдәҺдёҖдёӘзү№е®ҡзҡ„ Stage дёӯгҖӮ
  • agent
    pipeline иҝҗиЎҢзҺҜеўғпјҢеҰӮжһңе®ҡд№үеңЁжңҖдёҠйқўиЎЁжҳҺж•ҙдёӘ pipeline йғҪеңЁеҗҢдёҖдёӘ agent дёӯжү§иЎҢпјҢд№ҹеҸҜд»ҘеңЁжҜҸдёӘ Stage еқ—дёӯеҚ•зӢ¬жҢҮе®ҡпјҢдҪңз”ЁдәҺеҚ•дёӘ StageгҖӮ
