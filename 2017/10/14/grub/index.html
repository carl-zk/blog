Grub

зӣ®зҡ„пјҡеҲ¶дҪңдёҖдёӘLinuxе’ҢwindowsеҗҜеҠЁзӣҳгҖӮ

grubе®ҳзҪ‘
grubе‘Ҫд»Өдҝ®еӨҚзі»з»ҹ
еӨҡеҗҲдёҖеҗҜеҠЁзӣҳеҲ¶дҪңе·Ҙе…· WinSetupFromUSBгҖҗWinSetupFromUSBж•ҷзЁӢгҖ‘
MultiSystem вҖ“ Create a MultiBoot USB from LinuxгҖҗMultiSystemж•ҷзЁӢгҖ‘
GNU GRUBжҳҜдёҖдёӘзі»з»ҹејҖжңәеј•еҜјеә”з”ЁгҖӮ

е‘ҪеҗҚи§„еҲҷ

(fd0) : fd иЎЁзӨәfloppy diskпјҢ0 иЎЁзӨәdrive numberпјҢdriverеҸ·д»Һ0ејҖе§ӢгҖӮ
(hd0,msdos2) : hd иЎЁзӨәhard disk driveпјҢmsdos иЎЁзӨәpartition schemeпјҢ2иЎЁзӨәpartition numberпјҢ partition numberд»Һ1ејҖе§ӢгҖӮ
(hd0,msdos5) : 5иЎЁзӨәhd0зҡ„з¬¬дёҖдёӘйҖ»иҫ‘еҲҶеҢәгҖӮ(дёҖдёӘзі»з»ҹеҸӘиғҪжңү4дёӘдё»еҲҶеҢә)
(hd1,msdos1,bsd1) : BSDзі»з»ҹдёӯпјҢз¬¬дәҢдёӘзЎ¬зӣҳз¬¬дёҖдёӘsliceзҡ„вҖҷaвҖҷеҲҶеҢәгҖӮ
еҰӮдҪ•жҢҮе®ҡдёҖдёӘж–Үд»¶пјҹ
(hd0,msdos1)/vmlinuz : з¬¬дёҖдёӘзЎ¬зӣҳзҡ„з¬¬дёҖдёӘдё»еҲҶеҢәзҡ„vmlinuzж–Үд»¶гҖӮ

еҰӮдҪ•з”ЁGRUBеј•еҜјOS

GRUBжңүдёӨз§ҚдёҚеҗҢзҡ„еј•еҜјж–№ејҸпјҡзӣҙжҺҘеј•еҜје’Ңchain-loadпјҲеј•еҜјеҸҰдёҖз§Қboot loaderпјҢи®©е®ғеҺ»load OSпјүгҖӮ
MultibootжҳҜGRUBеҺҹз”ҹж”ҜжҢҒзҡ„пјҢе®ғеҸҜд»ҘзӣҙжҺҘеј•еҜјLinux, FreeBSD, NetBSD and OpenBSDзӯүзі»з»ҹгҖӮ
дёҚж”ҜжҢҒMultibootеҸҠGRUBжІЎжңүжҳҺзЎ®иЎЁзӨәж”ҜжҢҒзҡ„зі»з»ҹеҸӘиғҪйҖҡиҝҮchain-loadеј•еҜјгҖӮ

Chain-loading is only supported on PC BIOS and EFI platforms.
